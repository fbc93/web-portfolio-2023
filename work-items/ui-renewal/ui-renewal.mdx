imagesLayout_01_image: images-layout-01-image.jpg
imagesLayout_01_content: 훈련소 카페 가입은 사용자가 보고싶은 군인으로 등록한 훈련병 데이터와 연명부에 있는 데이터가 일치하는지 확인하는 절차입니다. 편지를 받을 훈련병의 데이터가 일치하는지 확인함으로써 그 후부터 사용자는 인터넷 위문편지를 작성할 수 있게됩니다. 핵심 서비스를 사용하기 위해 사용자가 거쳐야하는 가장 중요한 절차지만, 오래된 기존 UI에서 사용자의 여정이 헷갈리고 명확하지 않다는 앱스토어/구글플레이 리뷰를 적극 반영하여 시작하게된 프로젝트입니다.
imagesLayout_02_image: images-layout-02-image.jpg
imagesLayout_02_content: 기존에 Swiper로 하나씩 화면에 보이던 훈련소 리스트를 Swiper를 제거하고 펼쳐서 최대 4개까지 노출했습니다. 숨겨져 있던 요소를 펼쳐서 화면으로 도출하여 최대한 사용자가 눈으로 명확하게 확인할 수 있도록 작업했습니다. 기존 화면을 새롭게 퍼블리싱하면서 스타일을 컴포넌트화하고 함께 개발하는 동료들이 추후에 다른 작업에서 활용할 수 있도록 클래스의 네이밍 규칙을 정하는 등 최대한 작업 사항을 백로그나 인수인계서에 문서화하여 공유했습니다.
